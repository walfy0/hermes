# hermes

## what is hermes

https://en.wikipedia.org/wiki/Hermes

一个轻量级代理工具,纯golang编写,socks5 + tcp通信

## 启动方式
- 服务端sh ./proxy/build.sh
- 客户端sh ./build.sh
